El desarrollo de nuestra página web

Depto. De Programación

HTML, CSS Y JS

Para la realización de esta página web se utilizó el lenguaje de marcado HTML (HyperText Markup Language) y varias hojas de estilos CSS (Cascading Style Sheets; en español, "Hojas de estilo en cascada"). Para crear distintas funcionalidades en nuestra página web, utilizamos un lenguaje de programación llamado "Javascript". Es un lenguaje muy flexible y permite modificar la estructura de la página. Al inicio, tuvimos algunas dificultades al planear el diseño que tendría la página. Luego de una serie de toma de decisiones, optamos por crear el diseño que actualmente tiene la página. Mediante este formato, podemos presentar todos los datos necesarios, de una manera sencilla de comprender.

Código de la página

La página principal contiene un total de 528 líneas de código. Dentro de este código, en conjunto con otros lenguajes, se utilizaron varias funcionalidades para permitir la elaboración de una página más dinámica y atractiva para el público.
