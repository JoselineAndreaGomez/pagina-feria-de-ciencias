Proceso

Algunos estudiantes de nuestro grado tomaron el reto de diseñar una página web, que es la que estan utilizando en este momento, así como la programación necesaria para hacer que esta casa domótica tuviera un buen funcionamiento, aprendieron sobre el uso de la programación con "Arduino", Arduino es una plataforma de hardware y software de código abierto diseñada para facilitar la creación de proyectos electrónicos y la programación de microcontroladores. Consiste en una placa electrónica con un microcontrolador y una serie de pines de entrada y salida que permiten conectar sensores, actuadores y otros componentes electrónicos.

¿Qué más saber sobre arduino?

La placa Arduino se programa utilizando el lenguaje de programación Arduino, basado en Wiring y C/C++. Se utiliza un entorno de desarrollo integrado (IDE) que proporciona herramientas para escribir, compilar y cargar el código en la placa Arduino.
